Get started in minutes 几分钟即可上手

Installation安装说明

  1. Download the DMG and open it 下载 DMG 文件并打开
  2. Drag Tokei.app into the Applications folder 将 Tokei.app 拖入 Applications 文件夹
  3. If macOS blocks the first launch, open Terminal and run: sudo xattr -rd com.apple.quarantine /Applications/Tokei.app 如果首次打开被系统拦截，打开终端运行：sudo xattr -rd com.apple.quarantine /Applications/Tokei.app
  4. Launch Tokei — it appears in the menu bar and starts tracking automatically 启动 Tokei —— 它会出现在菜单栏，自动开始追踪

That's it. No account, no API key, no configuration needed. 就这么简单。无需账号、无需 API 密钥、无需任何配置。

Multi-device Sync多设备同步

  1. Create a private Git repository on GitHub (or any Git host) 在 GitHub（或其他 Git 平台）创建一个私有仓库
  2. Clone the repo to your Mac, then open Tokei → Settings → Multi-device Sync → Enable → select the repo folder 将仓库克隆到 Mac，打开 Tokei → 设置 → 多设备同步 → 开启 → 选择仓库目录
  3. On remote Linux servers: clone the same repo, install the collection script, set up a cron job for auto-sync every 5 minutes 远程 Linux 服务器：克隆同一仓库，安装采集脚本，配置 cron 每 5 分钟自动同步
  4. Each device collects data independently — Tokei merges them automatically 每台设备独立采集数据 —— Tokei 自动聚合展示
